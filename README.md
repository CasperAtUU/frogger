# Canal Hop

A small original browser arcade game: guide a frog across busy traffic and a lively canal to reach five home berths.

## Play

Use the arrow keys to hop:

- **↑** Up
- **↓** Down
- **←** Left
- **→** Right

On touch devices, swipe in the direction you want to hop.

Avoid traffic, stay on river platforms, watch for diving turtles, and do not linger too long—the hurry meter and the heron are both watching.

## Run locally

Download or clone this repository and open `index.html` in a modern browser.

## Publish with GitHub Pages

This repository is a static site. In GitHub, enable **Settings → Pages**, then publish from your default branch and the repository root. Your game will be available at:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/`

## Notes

- Built as one self-contained HTML file.
- No external libraries, assets, sound, or music.
- High score is stored in the browser with `localStorage`.
